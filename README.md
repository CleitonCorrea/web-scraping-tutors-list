# Web Scraping Tutors
### Esta aplicação realiza o scraping na página web, realiza o login com o usuário e dispara email para os alunos 

![Web Scraping]("public/images/logo-tutors.png")


### Up Ambiente

```
 docker-compose up -d
```
### Instal NPM
```
    npm instal
```
### Start a application

```
    npm start

```
