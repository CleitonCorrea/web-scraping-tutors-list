# Web Scraping Tutors
### Esta aplicação realiza o scraping na página web, realiza o login com o usuário e dispara email para os alunos 

![Web Scraping](./public/images/Puppeteer.png)


### Up Ambiente

```
 docker-compose up -d
```
### Instal NPM
```
    npm instal
```
### Start a application

```
    npm start

```

### Acesse:

```
    http://127.0.0.1:300

```
